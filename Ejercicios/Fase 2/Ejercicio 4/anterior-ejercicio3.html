Desarrollar un script que permita arrastrar y soltar imágenes que se abrirán en el interior de las zonas definidas. Si realizo dicha acción fuera de esas zonas, se debe cancelar el evento automático correspondiente. Así mismo, cada una de estas zonas permitirá seleccionar las recursos desde el file system. Los archivos de imágen deberán estar en la misma carpeta del proyecto.
